# 1cm³와 1m³ 체험하기 🧊

초등학교 6학년 수학 — **1세제곱센티미터(1cm³)와 1세제곱미터(1m³)의 관계**(1m³ = 1,000,000cm³)를
태블릿에서 직접 손가락으로 돌려보며 체험하는 인터랙티브 학습 프로그램입니다.

## 주요 기능
- 👆 **터치로 정육면체 돌리기** (태블릿·스마트폰·PC 모두 지원)
- 🧱 1cm³ → 한 줄(100) → 한 층(10,000) → **1m³(1,000,000)** 단계별 쌓기
- 🔍 **100 × 100 × 100 격자를 한눈에** 보고, 확대해서 칸 하나가 1cm³임을 확인
- 📦 크기 비교 + ✅ 확인 퀴즈

## 사용 방법
`index.html` 파일을 브라우저에서 열면 됩니다. 별도 설치·인터넷 연결 불필요.

## 인터넷에 배포하기 (GitHub Pages)
1. GitHub에서 새 저장소(repository)를 만듭니다. (예: `volume-cube`)
2. 아래 명령으로 업로드합니다. (`<사용자명>`, `<저장소>` 부분 교체)
   ```bash
   git remote add origin https://github.com/<사용자명>/<저장소>.git
   git branch -M main
   git push -u origin main
   ```
3. GitHub 저장소 → **Settings → Pages → Branch: main / (root) → Save**
4. 잠시 후 `https://<사용자명>.github.io/<저장소>/` 주소로 학생들이 접속할 수 있습니다.
   (이 주소를 QR코드로 만들어 나눠주면 태블릿에서 바로 열 수 있어요.)
